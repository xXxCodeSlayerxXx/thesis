This is a description of the historical structure of the place_box_list_branch_and_bound() function, and an archive of all parts of it that were removed or deprecated.

STRUCTURE (# for deprecated code):

function definition:
    pre-work:
        sort box list
        determine guarantee use
        init sequence storage
        init BFD best score
        estimate node count
    
    precalculations:
        precalc box orientation dict
        bounding rule precalculations:
            # precalc rule 2 footprint dict
            precalc rule 4 tallest remaining lookup list
        branching filter precalculations:
            precalc filter 2 dimension tuples list
            # init filter 3 dict
    
    recursive function definition:
        list nonlocal variables
        base case:
            count pallet score
            update best score and sequence if score < best_score
            return
        bounding cases:
            bounding rule 1 (trivial):
                check if intermediate state > best score
                register bounding and return
            # bounding rule 2 (volume):
                check whether box volume to go fits under current best score
                register bounding and return
            # bounding rule 3 (look-ahead):
                check if next box best case placement exceeds best score
                register bounding and return
            bounding rule 4 (tall-low):
                check if shortest-oriented tallest remaining box in best-case placement exceeds best score
                register bounding and return
        branching case:
            find next box, its orientations, and get list of sorted extpts
            init filter 1 profile set
            check if filter 2 is relevant and store key
            create list of placement candidates
            filter 5: (Top-X Limiting):
                score placement list by z-height and x, y coordinates
                filter candidate list to top X scores
                register filtering
            loop on candidate list:
                check placement validity
                # filter 4 (two-surface):
                    check if potential placement touches an edge or box on two sides
                    register filtering and continue
                filter 1 (deduplication):
                    check if use guarantee is off
                    check if candidate placement is similar (same orientation and height)
                    register filtering and continue
                filter 2 (symbreak):
                    check if symbreak key is registered and smaller than current tuple set
                    register filtering and continue
                place box
                # filter 3 (heightmap rec):
                    hash current heightmap at current depth
                    check if it exists in the dict or place it
                    register filtering, remove box, and continue
                add box to current sequence
                register node traversal
                recurse with index + 1
                remove box

    init prune and filter counts
    init node tqdm
    init pruning tqdms
    start recursive function
    reconstruct pallet from best sequence (or fail)
    compile and return bnb stats
